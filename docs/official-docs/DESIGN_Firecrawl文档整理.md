## 🎨 设计规范

### 文件命名规范
1. **目录命名**：`序号-中文名称/`（如：`01-快速开始/`）
2. **文件命名**：`功能描述.md`（如：`Scrape-单页抓取.md`）
3. **README文件**：每个目录都有README.md作为导航
4. **特殊标识**：使用emoji增强可读性

### 内容组织规范
1. **统一模板**：每个文档都遵循统一的结构模板
2. **交叉引用**：使用相对路径进行文档间引用
3. **代码示例**：每个功能都提供完整的代码示例
4. **多语言支持**：保持中英文对照
